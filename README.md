# 📊 Superstore Sales Dashboard (Power BI)

## 📌 Project Overview
This project presents an interactive Power BI dashboard built using the Superstore dataset to analyze sales performance and business insights.

## 🚀 Key Features
- KPI Metrics: Total Sales, Profit, Orders, Profit Margin
- Interactive Filters: Region, Category, Segment, Date
- Visualizations: Bar Chart, Line Chart, Pie Chart, Map, Scatter Plot

## 📊 Key Insights
- West region has the highest sales performance
- Technology category contributes maximum revenue
- Identified products with high sales but low profitability

## 🛠️ Tools Used
- Power BI
- Excel
- DAX
